Windows

Install Git Client

Suggestions:

  • SmartGit (for GUI)
  • GitSCM

Install Visual Studio 2019 with C++ Desktop Option

Set Environment Variable

__archive C:\__archive

 

Create folder at C named __archive and set it current:

mkdir C:\__archive

C:

cd \__archive

git clone https://github.com/ca2/storage-windows

git clone https://github.com/ca2/third-windows

git clone https://github.com/ca2/tool-windows

 

 

 

Create folder (#no spaces prefer) at some "nice place" and set it current:

 

mkdir C:\myca2-test

C:

cd \myca2-test

svn checkout https://ca2.io/shared/app/basis/ app

svn checkout https://ca2.io/shared/app-simple/basis/ app-simple

svn checkout https://ca2.io/shared/include/basis/ include

svn checkout https://ca2.io/shared/platform-windows/basis/ platform-windows

svn checkout https://ca2.io/shared/third/basis/ third

 

Open C:\myca2-test\platform-windows\_seed\simple.sln at Visual Studio
