How to build ca2 Framework simple solution at Windows...

 Check the prerequisites at Windows,
if you haven't done so already.

>

 Create folder named simple (replace simple by desired name) and set it current:

mkdir C:\<simple>

C:

cd \<simple>

git clone https://github.com/ca2/windows-simple-solution --recurse-submodules

 

 

Open C:\simple\archive\platform-windows\_seed\simple.sln at Visual Studio
