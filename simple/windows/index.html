How to build ca2 Framework simple solution at Windows...

Check the prerequisites at Windows, if you haven't done so already.

 Create folder named LocalFolderForTheCa2SimpleExampleSolution and set it current:

LocalFolderForTheCa2SimpleExampleSolution by desired name and set it current

mkdir C:\LocalFolderForTheCa2SimpleExampleSolution

C:

cd \LocalFolderForTheCa2SimpleExampleSolution

git clone https://github.com/ca2/windows-simple-solution --recurse-submodules

 

 

Open C:\LocalFolderForTheCa2SimpleExampleSolution\archive\platform-windows\_seed\simple.sln at Visual Studio
