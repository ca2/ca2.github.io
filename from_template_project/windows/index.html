How to create a ca2 Framework solution and project from a "template" project at Windows...

Our "template" project/app will be app_simple_drawing https://github.com/ca2/app-simple/drawing

Check the prerequisites at Windows, if you haven't done so already.

 Create repositories for SolutionUsingCa2:

Replace SolutionUsingCa2 by desired name for your solution in GitHub.

The given example here shows example for 3 (three) online repositories to be created at GitHub:

  • https://github.com/yourgithubaccount/SolutionUsingCa2
  • https://github.com/yourgithubaccount/SolutionUsingCa2-source
  • https://github.com/yourgithubaccount/SolutionUsingCa2-windows-solution

mkdir C:\LocalFolderForSolutionUsingCa2

C:

cd \LocalFolderForSolutionUsingCa2

git clone https://github.com/yourgithubaccount/SolutionUsingCa2-windows-solution .

 For windows development:

C:

cd \LocalFolderForSolutionUsingCa2

git submodule add https://github.com/ca2/windows-archive archive

 Create source submodule for platform neutral projects:

The source folder will contain files that are platform neutral (or mostly neutral)

C:

cd \LocalFolderForSolutionUsingCa2

git submodule add https://github.com/yourgithubaccount/SolutionUsingCa2-source source

 Add the platform neutral submodules

Add the ca2 platform neutral projects and your "SolutionProjectsUsingCa2"

C:

cd \LocalFolderForSolutionUsingCa2\source

git submodule add https://github.com/ca2/app app

git submodule add https://github.com/ca2/include include

(*Source of Template Project Using ca2) git submodule add https://github.com/ca2/app-simple app-simple

* For projects that would server as sample/example/template/model.

git submodule add https://github.com/ca2/third third

git submodule add https://github.com/yourgithubaccount/SolutionUsingCa2 SolutionProjectsUsingCa2

 Create folder structure for your Application1 project

Replace Application1 by the name of your app project at SolutionUsingCa2

Copy folder structure from C:\LocalFolderForSolutionUsingCa2\source\app-simple\_matter\drawing to C:\LocalFolderForSolutionUsingCa2\source\SolutionUsingCa2\_matter\Application1

https://github.com/yourgithubaccount/SolutionUsingCa2/_matter/Application1/_std/_std (for language neutral or default resources)

https://github.com/yourgithubaccount/SolutionUsingCa2/_matter/Application1/en-us/en-us (for language english resources)

...

Copy folder structure from C:\LocalFolderForSolutionUsingCa2\source\app-simple\drawing to C:\LocalFolderForSolutionUsingCa2\source\SolutionUsingCa2\Application1

https://github.com/yourgithubaccount/SolutionUsingCa2/Application1 (for source and project files for Application1 project)

The Visual Studio solution file is recommended to be put in path below:

C:\LocalFolderForSolutionUsingCa2\source\SolutionProjectsUsingCa2\_seed\Application1.sln

Remote path: https://github.com/yourrepository/SolutionUsingCa2/_seed/Application1.sln
