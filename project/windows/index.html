&nbps;How to Build ca2 Framework at Windows

 Check the prerequisites at Windows,
if you haven't done so already.

>

 Create repository named project (replace project by desired name) and set it current:

The given example here shows example for 3 (three) online repositories previously created at GitHub:

    https://github.com/example/project-windows-solution https://github.com/example/project-source https://github.com/example/project

mkdir C:\project

C:

cd \project

https://github.com/example/project-windows-solution

 For windows development (with command prompt folder C:\project):

git submodule add https://github.com/ca2/windows-archive archive

 Your project "source" and ca2 Framework common source will be here (with command prompt folder C:\project):

git submodule add https://github.com/ca2/app app

git submodule add https://github.com/ca2/include include

(Optionally for example projects) git submodule add https://github.com/ca2/app-simple app-simple

git submodule add https://github.com/example/project project

The Visual Studio solution file is recommended to be put in path below:

C:\project\source\project\_seed\project.sln
